Just connect one wire of a button a ground and the other wire to a action pin directly above it with a pico. This project needs the following:
(https://github.com/user-attachments/files/29010785/adafruit-circuitpython-hid-10.x-mpy-6.1.10.zip) and the newest version of curcuit python
This is basically just a button that presses 'v' when it is pressed, in the files please use rectaliniear infill patern with 15% infill so that you can open up the plunger to thread the button wires through it to make sure that the pico can be outside the case (As show in the picture). Just to add I used ai for the small amount of code in this project.

Overall I hope you really enjoy my project, it is super funny when it works and it has an interesting 2 player style when 2 players share a controler and one gets the mega stimm.
